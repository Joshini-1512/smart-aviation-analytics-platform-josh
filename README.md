# Smart Aviation Operations & Passenger Analytics Platform

## Project Overview
This project is developed using Snowflake, AWS S3, and Power BI to build a scalable aviation analytics platform.

## Architecture
RAW → CORE → MART

## Features
- Passenger Analytics
- Revenue Analytics
- Flight Delay Monitoring
- Airline Performance Dashboard
- Streams & Tasks Automation
- Dynamic Tables
- Snowpipe
- RBAC Security
- Masking Policies
